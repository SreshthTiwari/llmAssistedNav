# Sreshth Tiwari PA4

In this project I've implemented an Occupancy Grid Mapping node in ROS 2 using:
- LiDAR scans
- TF transforms
- Bresenham raycasting
- probabilistic log-odds updates (extra credit)

---

## Required Topics and Frames

This mapper assumes the following ROS interfaces are available:

### Topics
- `/base_scan` — `sensor_msgs/LaserScan`
- `/cmd_vel` — `geometry_msgs/Twist`
- `/map` — `nav_msgs/OccupancyGrid`

### TF Frames
- `rosbot1/odom`
- `rosbot1/base_link`

These parameters can be updated in `pa4.py`.

---

## Setup

### 1. Run the python file
Open a terminal and run:

```bash
python3 pa4.py
```

### 2. Open Rviz2
In a new terminal, run:

```bash
rviz2
```

This opens up the rviz window on vnc, where the map generated by the node can be visualized.
